About This Project

A high-performance search engine built from the ground up.

This application is the result of the designed to build a complete The goal was to implement core information retrieval concepts and build a modern, responsive user interface.

Technology Stack:
  • Backend: Python with Flask, NLTK for natural language processing, Pandas for data handling.
  • Core Logic: Inverted Index for fast document retrieval and TF-IDF for ranking search results.
  • Frontend: HTML, CSS, and vanilla JavaScript, styled with the Bootstrap 5 framework.

Every feature, from the backend data pipeline to the interactive frontend shortcuts, was built to demonstrate a practical understanding of how modern search technology works.
