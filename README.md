# Advanced VR Search Engine

A Flask-based search engine implementing Information Retrieval concepts including Inverted Index and TF-IDF ranking.

---

## Features

- Fast document retrieval using Inverted Index
- TF-IDF ranking
- Query preprocessing with NLTK
- Responsive Bootstrap UI
- Shortcut Management
- About & Contact Pages
- Flask REST API

---

Flask
NLTK
Bootstrap
